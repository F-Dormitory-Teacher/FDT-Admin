# FDT-Admin

## 설명

- 코로나 사태, 기숙사 학교에서 외부와 가장 접촉이 많은 기숙사 선생님들로부터 학생들을 안전하게 보호하기 위한 무인 기숙사 관리 시스템
- 2020 SW 마이스터고 연합 해커톤 베어런 대표 이사상
